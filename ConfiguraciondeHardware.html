Paso 3: Configuración de la Red

  1. Configuración de la Red Interna: Asigna direcciones IP estáticas a todos los nodos de cómputo y al nodo de cabeza.
  2. SSH Sin Contraseña: Configura el acceso SSH sin contraseña desde el nodo de cabeza a todos los nodos de cómputo.
    3. ssh-keygen -t rsa
    ssh-copy-id user@compute-node

Paso 4: Instalación de las Dependencias

  1. Instala los Compiladores y MPI

    En cada nodo, instala GCC y una implementación de MPI, como OpenMPI o MPICH.

    sudo apt-get update sudo apt-get install -y build-essential gcc gfortran openmpi-bin openmpi-common libopenmpi-dev

  2. Instala NetCDF y HDF5

    Instala las bibliotecas NetCDF y HDF5 necesarias para WRF..

    sudo apt-get install -y libnetcdf-dev libnetcdff-dev libhdf5-serial-dev
